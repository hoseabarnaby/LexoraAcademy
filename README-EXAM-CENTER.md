# Lexora Exam Center

Tambahan latihan orisinal Lexora untuk English IELTS, Mandarin HSK, Japanese JLPT, dan German CEFR/Goethe.

## Tipe latihan
- True/False/Not Given
- Yes/No/Not Given
- Multiple Choice
- Matching
- Matching Headings
- Sentence/Summary Completion
- Pinyin dan Kanji Reading
- Word Order
- Translation
- Grammar/Error Correction
- Listening simulation
- Writing task
- Speaking prompt

Materi ini mengikuti gaya ujian tetapi bukan soal resmi dan tidak menyalin materi Cambridge atau lembaga ujian lain.
