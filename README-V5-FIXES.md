# Lexora Academy V5 — Polished & Performance Fix

Perbaikan utama:

- Adventure/karakter kembali memakai stylesheet lengkap dan responsive.
- Exam Center selalu mengikuti bahasa aktif: IELTS, HSK, JLPT, atau Deutsch.
- Bank exam diperluas dengan Reading, Listening, Grammar, Writing, dan variasi soal orisinal.
- Kosakata besar dimuat per bahasa saat dipilih, bukan 4 bahasa sekaligus.
- Sesi login tersimpan tidak lagi menampilkan halaman login 5–10 detik.
- Login Google dihapus.
- Cloud sync berjalan di belakang tanpa mengunci halaman.
- Mobile layout diringkas menjadi satu kolom dengan tombol yang mudah disentuh.

## Konfigurasi
Cari `KONFIGURASI LEXORA` di bagian bawah `index.html`. Isi Supabase URL, anon/publishable key, dan OpenRouter key. Jangan memasukkan service role ke website user.
